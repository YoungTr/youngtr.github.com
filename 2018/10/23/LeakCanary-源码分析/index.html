1. 内存泄漏

  • 集合类泄漏；
  • 单例造成的内存泄漏；
  • 匿名内部类/非静态内部类创建静态实例被 Activity（context）持有；
  • Handler 造成的内存泄漏，退出页面 message 未完成，使用 static handler 或者 activity 弱引用；
  • 线程造成内存泄漏，页面退出任务未结束；
  • WebView 造成内存泄漏，可以在单独进程中运行，退出之后直接 kill process；
